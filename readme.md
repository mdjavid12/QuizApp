# Quiz App API

A backend API for quizzes using **FastAPI + SQLAlchemy + JWT**. Supports user registration, login, admin CRUD for questions, and quiz submission with scoring.

---



## Installation

```bash
# Clone repo
git clone <repo-url>
cd QuizApp

# Create virtual environment
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt


# Swagger UI
http://127.0.0.1:8000/docs
